# Extend Everything

This is a sample repository that sets up a few Umbraco ~~15~~ ~~15~~ 17 back office
extensions that tries to do a bit of everything.

Sets everything up with Aspire and both npm and yarn based backoffice projects.

A greate sample password and username would be `jack.sparrow@pirates.com` and `password123`.

## Tetris Dashboard

Adds a custom dashboard that features a playable Tetris game. Submitted to the
[Umbraco Marketplace](https://marketplace.umbraco.com/package/extendeverything.tetrisdashboard)
both for fun and as a demonstration of how to create and publish a backoffice
extension.
